Components
Frontend — React + Vite (port 5173)
Dashboard.jsx — Health score + activity feed
SymptomChecker.jsx — Symptom form → API call → results
HealthTracker.jsx — Vitals form + Chart.js charts
Chatbot.jsx — Chat UI with Flask chatbot endpoint
Backend — Flask (port 5000)
POST /api/symptoms/analyze — ML symptom prediction
POST /api/chat — Chatbot intent response
GET/POST /api/vitals — SQLite vitals storage
GET /api/model/info — Model metadata
Python AI Model — scikit-learn
Algorithm: TF-IDF Vectorizer + Random Forest Classifier
Training data: 600+ symptom-condition records (18 conditions)
Output: Top conditions + probabilities + urgency + recommendations
Saved as: symptom_model.pkl (joblib)
